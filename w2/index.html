Deployment Notes

This app runs entirely in the browser with TensorFlow.js. To deploy on GitHub Pages:

  1. Create a public GitHub repository
  2. Commit index.html and app.js to the repository
  3. Go to repository Settings → Pages → Source → select "main" branch, root folder
  4. Your app will be available at: https://[your-username].github.io/[repository-name]
  5. Check browser console for CSV parsing/feature gate debug information

Note: For other datasets, swap the schema in app.js (target: Survived, features: Pclass, Sex, Age, SibSp, Parch, Fare, Embarked).
