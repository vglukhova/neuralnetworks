Deployment Notes

To deploy this application on GitHub Pages:

  1. Create a public GitHub repository
  2. Commit both index.html and app.js to the repository
  3. Go to repository Settings → Pages → Source: select "main" branch and root folder
  4. Click Save and wait for deployment (typically takes 1-2 minutes)
  5. Your app will be available at: https://[your-username].github.io/[repository-name]

Note: This app runs entirely in the browser using TensorFlow.js. No server-side processing is required.
